## Project Overview
This project visualizes racial disparities in neurodegenerative disease diagnoses. It utilizes a matched case-control design to highlight systemic inequities, specifically comparing the representation of Black and White patients across various conditions.

## Dataset Note
Due to patient privacy regulations and HIPAA compliance, the raw clinical dataset (approximately 10,500 records from the Emory Healthcare system, 2015-2024) is strictly confidential and not included in this public repository. All code provided operates on the assumption of local data access, and the final dashboard displays only aggregated, de-identified metrics.
